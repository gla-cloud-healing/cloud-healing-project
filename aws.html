Overview
AWS Services Used
🖥️
EC2
Virtual machine that runs Docker, Kubernetes, Flask, Prometheus and Grafana. t2.micro is free for 12 months.
🔒
Security Groups
Firewall rules — open ports 22 (SSH), 5000 (Flask), 9090 (Prometheus), 3000 (Grafana), 80 (HTTP).
🌐
Public IP
EC2 assigns a public IPv4 address — use it to access Flask app and dashboards from browser.
🔑
Key Pairs
.pem file used for SSH access. chmod 400 to secure it before connecting from terminal.

Code
EC2 Setup Commands
Terminal — After SSH into EC2 
# Connect via SSH (GitBash on Windows)
ssh -i "your-key.pem" ubuntu@YOUR_PUBLIC_IP

# Update system
sudo apt update && sudo apt upgrade -y

# Install Docker
sudo apt install -y docker.io docker-compose
sudo systemctl start docker
sudo systemctl enable docker
sudo usermod -aG docker ubuntu

# Clone project
git clone https://github.com/gla-cloud-healing/cloud-healing-project.git
cd cloud-healing-project

# Deploy full stack
docker-compose up -d

# Access from browser:
# Flask:      http://YOUR_PUBLIC_IP:5000
# Prometheus: http://YOUR_PUBLIC_IP:9090
# Grafana:    http://YOUR_PUBLIC_IP:3000
